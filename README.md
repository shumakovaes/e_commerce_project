# Выявление профилей потребления покупателей интернет-магазина

Аналитический проект по данным интернет-магазина товаров для дома «Пока все ещё тут». Цель проекта — сегментировать покупателей по покупательскому поведению и подготовить рекомендации для маркетинговой команды, чтобы повысить повторные покупки, удержание клиентов и выручку.

## Задача

На основе транзакционных данных определить, какие группы покупателей есть у магазина, чем они отличаются по частоте покупок, среднему чеку, товарным предпочтениям и сезонной активности. По итогам анализа сформулировать практические рекомендации для коммуникаций с каждым сегментом.

## Данные

В работе использован датасет `ecommerce_dataset.csv` с транзакциями за период с 1 октября 2018 года по 31 октября 2019 года.

Поля датасета:

- `date` — дата и время заказа;
- `customer_id` — идентификатор покупателя;
- `order_id` — идентификатор заказа;
- `product` — наименование товара;
- `quantity` — количество товара;
- `price` — цена товара.

## Стек

`Python`, `pandas`, `numpy`, `matplotlib`, `plotly`, `scipy`, `statsmodels`, RFM-сегментация, проверка статистических гипотез, визуализация данных.

## Что сделано

- Проведена предобработка данных: преобразование дат, нормализация названий товаров, проверка пропусков, явных и неявных дубликатов.
- Удалены конфликтующие и повторяющиеся записи, а также один экстремальный заказ на 1000 единиц товара.
- Рассчитана выручка по каждой строке заказа.
- Товары вручную распределены по 5 категориям: товары для дома, цветы и растения, посуда и аксессуары, хранение/сумки/тележки, строительные товары.
- Проанализирована динамика выручки, количества заказов и среднего чека.
- Проведена RFM-сегментация покупателей.
- Проверены статистические различия между сегментами по среднему чеку и частоте заказов с поправкой Холма.
- Сформулированы рекомендации для маркетингового отдела.

## Ключевые результаты

После очистки в анализе осталось 4 734 строки, 2 742 заказа и 2 383 покупателя. Общая выручка составила 3 385 607.

Выручка по категориям:

| Категория | Выручка |
|---|---:|
| Товары для дома | 1 507 760 |
| Хранение, сумки и тележки | 830 289 |
| Цветы и растения | 552 168 |
| Посуда и аксессуары | 367 886 |
| Строительные товары | 62 097 |

Сегменты покупателей:

| Сегмент | Клиентов | Заказов | Средний чек |
|---|---:|---:|---:|
| Спящие | 1 187 | 1 406 | 1 299.32 |
| Потенциальные | 599 | 674 | 1 263.70 |
| Новые | 567 | 583 | 1 040.62 |
| Лояльные | 30 | 79 | 1 270.18 |

Основные наблюдения:

- Самая большая доля клиентов находится в сегменте спящих покупателей.
- Лояльных клиентов мало, но они совершают в среднем больше заказов.
- Новые клиенты имеют самый низкий средний чек, поэтому для них особенно важны механики повторной покупки и роста корзины.
- В категории «Цветы и растения» заметна сезонность: активность растёт весной.
- Наибольшую выручку приносят товары для дома, поэтому их стоит использовать как базовую категорию для регулярных коммуникаций.
- Статистически значимые различия по среднему чеку найдены между сегментами «лояльные — новые» и «спящие — новые».
- По частоте заказов статистически значимые различия обнаружены между всеми парами сегментов.

## Рекомендации

**Лояльные клиенты:** развивать программу лояльности, персональные предложения и сезонные акции. Для этой группы особенно релевантны цветы и растения, поэтому стоит тестировать расширение ассортимента и специальные подборки в сезон.

**Новые клиенты:** стимулировать вторую покупку с помощью приветственных цепочек, скидок на следующий заказ, предложений на товары для дома, посуду и растения. Для роста среднего чека можно использовать пороговые скидки и комплекты товаров.

**Потенциальные клиенты:** поддерживать контакт между сезонными всплесками. Весной делать акцент на цветах и растениях, в остальные периоды продвигать всесезонные товары для дома.

**Спящие клиенты:** запускать реактивационные кампании с персональными скидками, бонусами за возвращение и короткими опросами о причинах неактивности. Для коммуникаций использовать категории, которые были популярны у сегмента ранее: товары для дома, посуду, цветы и растения.
